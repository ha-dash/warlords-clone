# Бриф Проекта

## 1. Обзор

**Warlords Clone** — это современная веб-версия классической пошаговой стратегии Warlords, реализованная с использованием HTML5, CSS3, JavaScript и Three.js. Проект представляет собой фэнтезийную стратегию, где игроки управляют армиями и героями, захватывая города и сражаясь за контроль над континентом.

Игра сохраняет оригинальный геймплей с улучшенной графикой (2D Canvas и 3D Three.js рендеринг), современным пользовательским интерфейсом на базе Shadcn/ui и редактором карт для создания пользовательских карт.

## 2. Ключевые Цели

1. **Воссоздание классического геймплея**: Сохранение оригинальной механики пошаговой стратегии Warlords
2. **Современная визуализация**: 3D рендеринг с использованием Three.js и реальных 3D моделей из assets
3. **Редактор карт**: Полнофункциональный 3D редактор карт с поддержкой размещения зданий и изменения местности
4. **Модульная архитектура**: Четкое разделение ответственности между компонентами для поддержки и расширения
5. **Миграция на современный стек**: Постепенный переход на Next.js 16 + React 19 (начиная с редактора карт)

## 3. Границы Проекта (Scope)

### Входит в объем (In-Scope)

- Пошаговая стратегия с гексагональной картой
- Система юнитов и героев с различными способностями
- Система городов и их развития
- Боевая система (автоматические сражения)
- ИИ противники
- Система заклинаний и предметов
- Сохранение/загрузка игры
- Редактор карт с 3D визуализацией
- 3D рендеринг игровой карты (Three.js)
- Использование реальных 3D моделей из папки assets/

### Не входит в объем (Out-of-Scope)

- Мультиплеер (онлайн кооператив/PvP)
- Облачные сервисы для сохранений
- Мобильная версия (только десктоп веб)
- Система достижений и статистики
- Редактор кампаний (только редактор карт)

## 4. Критерии Успеха

1. Игра полностью играбельна в браузере без дополнительных плагинов
2. Редактор карт позволяет создавать и сохранять пользовательские карты
3. 3D рендеринг работает с приемлемой производительностью (30+ FPS)
4. Код покрыт тестами для критических компонентов
5. Документация актуальна и синхронизирована с кодом (Memory Bank)

## 5. Технологические Ограничения

- **Браузерная игра**: Работает в современных браузерах без серверной части
- **Локальные данные**: Все сохранения в LocalStorage браузера
- **Статические assets**: Модели и текстуры хранятся локально в папке assets/
- **Vanilla JS основной проект**: Пока не переходит на React (кроме редактора карт)
